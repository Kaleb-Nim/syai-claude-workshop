# Verify the install

Ask Claude:

```
verify playwright-cli is installed and show me the version
```

Expect something like:

```bash
$ playwright --version
Version 1.x.x
```

<div class="text-dim text-sm mt-4">
  If it errors out — paste the error straight back to Claude. It will diagnose PATH issues, missing browsers, or permission errors and fix them.
</div>

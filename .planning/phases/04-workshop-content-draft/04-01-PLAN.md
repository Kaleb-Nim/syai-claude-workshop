Slidev `src:` directive shape (from sli.dev/guide/syntax#importing-slides):
- A slide separator `---` followed immediately by a frontmatter block whose only key is `src:`
- The src target is a markdown file, path relative to the file containing the directive
- The included file may itself contain multiple `---`-separated slides — they all flatten in
- Per-slide frontmatter inside the included file (e.g. `layout: section`) is honored
- Example shape:
  ```
  ---
  src: ./pages/00-intro.md
  ---
  ```
